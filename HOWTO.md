# No Way But Down
* Requirements
    * Debian Wheezy/Ubuntu 12.04+
    * Windows 7 and MINGW

## Commands

* Movement: Arrow Keys or IJKL
* Menu: Escape
* Grab: g
* Drop: d (From Menu Only)
* Wield: w (From Menu Only)
* Use: Enter (From Menu Only)
* Quit: Close Window
* Help Screen: ?
